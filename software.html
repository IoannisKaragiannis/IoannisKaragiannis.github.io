Software

Here you can find all software implementations I have been working with in my spare time. The software is not maintained.

  • LinearAlgebra - This was a project I worked with in the fall of 2017. My intention was to practise my skills in C++ and in particular in C++14. This is an easy-to-install linear algebra library. It consists of two major classes, Vec standing for vector and Mat standing for matrix together with a set of accompanying functions and operations. This library might be useful for projects within the field of signal/image processing, machine learning, and in general in any kind of academic/scientific project.
